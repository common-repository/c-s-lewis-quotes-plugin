== Description ==

This a plugin, similiar to the well known dolly-plugin. It gives you 
randomly quots from C.S. Lewis, the man behind the Narnia chronicles.

== Installation ==

1. Upload `lewis_quotes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
